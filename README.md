# Iyan-Kim

A compact single-page personal site for links and portfolio work at `studio.iyan-kim.dev`.

## Structure

- `index.html`: Home hero, portfolio grid, SEO/social metadata
- `styles.css`: Dark theme, navigation, responsive layout, card states
- `script.js`: Active navigation indicator and portfolio reveal animation
- `assets/favicon.svg`: Site favicon
- `assets/og-image.png`: Open Graph / share thumbnail
